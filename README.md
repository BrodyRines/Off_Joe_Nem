#CS Market Project

A stock market app that tracks the market for Counter Strike

##Summary

We are using the csfloat API and the Steam API to get market and personal inventory information. From that we can show how much your in game items have changed in value over a specified amount of time, like a real stock market. It can also show you the total value of your inventory, and we plan to add a full GUI for the program.

###Dependencies

We are using an online library called "requests", which needs to be installed before the program can be used

###Installing

Before use, you need to aquire your own csfloat API key, which can be done at this link: https://csfloat.com/profile
Once aquired, you must create your own .env file in the main folder. once there, we must format the file like so:

<!-->

[MY_KEY]  
KEY=' <CSFLOAT API KEY> ' <--These 2 lines are all you need, formatted like so
-->
